Samurai Horse Revenge is a fast paced, hack and slash wave survival game created in ~48 hours for the Global Game Jam 2024. You play as the last standing samurai horse in a war against humanity in feudal japan.

This was my second ever game jam, and I greatly enjoyed working on it in the fast paced environment, with some great team members. I learned a lot implemeintg an input buffering system, which enabled the player to perform combos and input cancels when attacking and dodging. This feature was crucial in ensuring the combat was snappy and flowed well. Tweaking the combat and implementing the animations in-engine was also great practice for refining user experience!

It was a great experience and I look forward to participating in the next Global Game Jam!
