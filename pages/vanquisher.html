Engine/Skills Used: MonoGame, C#, Game Engine Programming

This project is a Work In Progress!

Vanquisher is a singleplayer 2D realtime strategy (RTS) game built around a unique ASCII Art visual style, and the simplicity of using only a single type of unit; the Soldier. The game is currently a work in progress project I started to build on my knowledge of lower level engine programming concepts, such as coordinate systems, collision detection, sprite rendering, and game logic optimization.

MonoGame is a free open source C# framework based on XNA, used for developing games. I chose it because I wanted to gain some experience implementing basic game engine features; I have so far implemented my own coordinate system, sprite rendering, and a camera class, a generic "GameObject", as well as derived "World Features", "Entities", and game loop logic for making these GameObjects do things!

With the foundational code for the project mostly completed, the player can currently only command and move existing units of soldiers. In future I plan to implement combat between opposing teams, resource gathering for purchasing new soldiers, advanced unit commands such as different formations, and finally Enemy AI.
